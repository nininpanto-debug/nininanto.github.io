Every future page, design decision, animation, story, image, video, and feature must follow this guide.

This document should only change when a major decision is made.

---

# Website Vision

This is NOT a portfolio.

This is NOT a blog.

This is NOT a business website.

This is a digital world built around one person.

The visitor should never feel like they are browsing a website.

They should feel like they are entering a completely different world.

Every click should encourage exploration.

Every page should create curiosity.

Every story should create emotion.

Every visitor should leave knowing who Ninin Anto is.

---

# Mission

Build the most immersive personal website possible.

The website should become the first destination when someone searches:

"Ninin Anto"

Visitors should not need to search anywhere else.

Everything about Ninin Anto should exist inside this world.
