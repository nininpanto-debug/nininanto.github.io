# 🌍 The World of Ninin Anto

> **One Person. Many Missions. Endless Possibilities.**

Welcome to the official repository of **The World of Ninin Anto**.

This project is not just a personal website—it's an immersive digital experience that tells the story of a life, a journey, dreams, failures, achievements, creativity, business, acting, writing, and continuous growth.

---

# 📖 About the Project

The World of Ninin Anto is designed as a living digital world instead of a traditional portfolio.

Visitors won't simply browse pages.

They will explore different "worlds," each representing an important part of Ninin Anto's journey.

Every world has its own identity, atmosphere, stories, visuals, and purpose.

This website will continue evolving throughout life.

# 🎨 Design Philosophy

The website follows five core principles:

- Story
- Emotion
- Experience
- Creativity
- Legacy

Every page should make visitors feel they are exploring a living world rather than browsing a website.

---

# 📱 Mobile-First Development

This project is primarily developed using **vscode.dev** on a mobile device.

To keep development manageable:

- Build complete files rather than partial edits.
- Keep the structure clean and organized.
- Use reusable components.
- Prioritize responsive design from the beginning.
